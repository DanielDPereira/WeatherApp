# WeatherApp

A simple, responsive weather application built with HTML, CSS, and vanilla JavaScript. It uses the **[Open-Meteo](https://open-meteo.com/)** public API — no API key required.

## Features

- 🔍 Search current weather by city name
- 🌡️ Display temperature with **°C / °F** toggle
- 💧 Humidity, 💨 wind speed, 🌡️ feels-like temperature, and 👁️ visibility details
- 🌤️ WMO weather-code descriptions and emoji icons
- Responsive layout — works on desktop and mobile
- Elegant dark glassmorphism UI

## Usage

Simply open `index.html` in any modern browser. No build step or server required.

## Tech Stack

| Layer      | Technology |
|------------|------------|
| Markup     | HTML5      |
| Styling    | CSS3 (custom properties, grid, animations) |
| Logic      | Vanilla JavaScript (ES2020+) |
| Weather API | [Open-Meteo Forecast API](https://open-meteo.com/) |
| Geocoding  | [Open-Meteo Geocoding API](https://open-meteo.com/en/docs/geocoding-api) |
